一些总结：

1. const_set 和 const_get 操作常量的用法

2. 举例子说明：动态方法的好处，减少代码冗余

3. %Q{} 语法使用，表示一行字符串

4. class_variables，class_variable_defined? 和 get, set 方法的使用

5. remove_const 和 const_set 对常量的动态操作

6. 适合迭代中，对每一个其中的元素做一次 Symbal 对象对应的方法调用。
