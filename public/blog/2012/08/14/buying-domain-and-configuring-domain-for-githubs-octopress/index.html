购买个人域名以及为部署在 GitHub 的Octopress 配置个人域名

Aug 14th, 2012 | Comments

部署到 GitHub 的 Octopress 默认采用二级域名 ＋ 二级目录来访问，如： http://blogbin.github.com/tech-blogbin-blog

现计划启用个人独立二级域名直接访问，即： http://tech-blog.blogbin.info/

一. 调整过程：

参阅： http://octopress.org/docs/deploying/github/

二. 问题

1. 404 错误：There isn’t a GitHub Page hosted here, sorry.

主要原因是没有设置正确的 source/CNAME，或者域名解析设置错误，或者耐心等待一段时间。

2. 加载错误的 CSS 路径

首页 index.html 中仍要加载 tech-blogbin-blog 子目录的 css 文件，正确调整第 4 点的 Octopress 配置文件可解决问题。
